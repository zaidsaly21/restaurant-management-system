# Restaurant Management System

A Java-based restaurant management system developed as an academic project to manage menu items, customer orders, payment processing, checkout, and discount handling.

## Project Overview

This project demonstrates the use of Object-Oriented Programming concepts, modular programming, SOLID principles, and common design patterns in Java. The system was developed to improve restaurant order handling and checkout operations through a structured Java application.

## Features

- Menu management
- Order management
- Payment processing
- Checkout and bill calculation
- Discount handling
- Receipt formatting
- Modular structure using packages
- Support for different discount strategies

## Technologies Used

- Java
- NetBeans IDE
- Object-Oriented Programming
- SOLID Principles
- Design Patterns

## Design Patterns Used

- Singleton Pattern
- Factory Method Pattern
- Strategy Pattern

## SOLID Principles Applied

- Single Responsibility Principle for separating responsibilities such as receipt formatting and order handling
- Open/Closed Principle for adding new discount policies without changing existing core logic

## Project Purpose

The purpose of this project is to practice Java programming, object-oriented design, clean code structure, and software design principles through a real-world restaurant management scenario.

## How to Run

1. Download or clone this repository.
2. Open the project in NetBeans IDE.
3. Run the project from the IDE.
